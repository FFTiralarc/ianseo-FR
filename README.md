# ianseo — Set français (FR)

Règles et configurations préconfigurées pour les compétitions françaises dans [ianseo](https://www.ianseo.net/).

Ce dépôt correspond au dossier `htdocs/Modules/Sets/FR/` de l'installation ianseo. Il est **directement intégré dans ianseo** et **mis à jour avec chaque nouvelle version d'ianseo**. Il n'est pas un module externe : aucune installation spécifique n'est requise.

Nous le publions ici pour la transparence, pour permettre à la communauté de suivre les évolutions et de comprendre les configurations retenues par la FFTA.

---

## Contenu

Ce set définit les règles applicables aux compétitions françaises pour les formats suivants.

### Tir extérieur — 70m (type 3)

| Règle | Compétition |
|-------|-------------|
| SetFRTAE | Championnat Elite |
| SetFRTAE-Valides | Sélectif |
| SetFRTAE-Para | Sélectif valides + para |
| SetFRChampionshipJun | Championnat Jeune individuel |
| SetFRChampJunTeams | Championnat Jeune par équipe |
| SetFRCoupeFrance | Coupe de France |
| SetFRChampsTNJ | Trophée National Jeune (TNJ) |
| SetFRD12023 | Division 1 — format 2023 |
| SetFRD12026 | Division 1 — format 2026 |
| SetFRFinDRD2 | Finales des DR |
| SetFRFinalsD2 | Finales D2 |

### Tir en salle — 18m (type 6)

| Règle | Compétition |
|-------|-------------|
| SetFrSelectif | Sélectif |
| SetFrSelectifPara | Sélectif valides + para |
| SetFRChampionshipSen | Championnat de France Adulte |
| SetFRChampionshipJun | Championnats de France Jeune |

### 25m et 25+18m (types 7 et 8)

| Règle | Compétition |
|-------|-------------|
| SetFrSelectif | Sélectif |
| SetFrSelectifPara | Sélectif valides + para |

### Beursault / Bouquet (type 50)

| Règle | Compétition |
|-------|-------------|
| SetFrBouquet | Bouquet provincial |
| SetFrBeursault | Beursault |

Le format Beursault inclut également un sous-module de saisie de scores dédié (`Modules/Bslt`), accessible dans le menu ianseo quand ce format est sélectionné.

---

## Mises à jour

Ce dépôt est synchronisé avec les évolutions du set FR intégrées dans ianseo. **Il n'y a rien à installer séparément** : le set est fourni avec votre installation ianseo standard.

Pour signaler une erreur dans les règles ou proposer une évolution, ouvrez une **issue** sur ce dépôt.
